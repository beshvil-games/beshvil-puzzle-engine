# דף ניהול המשחק

יש להעלות את כל התיקייה `admin` לתיקייה הראשית של המאגר:

`beshvil-puzzle-engine/admin/`

לאחר ההעלאה, דף הניהול יהיה בכתובת:

https://beshvil-games.github.io/beshvil-puzzle-engine/admin/

## מה הדף עושה

- שינוי סדר החידות בגרירה או באמצעות החיצים
- שינוי שם כל תחנה
- כיבוי חידות שלא משתתפות במשחק
- שמירת הסדר בדפדפן
- יצירת קוד QR לכל חידה פעילה
- הורדת כל קוד כקובץ PNG
- הדפסת כל הקודים או שמירתם כ-PDF

## חשוב

הודעת הסיום נשמרת כרגע במסך הניהול בלבד.
היא עדיין אינה משנה אוטומטית את הודעת הסיום של החידה האחרונה.
